== Shop Lab Backend

This is a backend written on Ruby on Rails stack for an example Shop Lab app which allows you to create and share shopping lists with your girlfriend/wife/soulmate.
See frontend counterpart written in Ionic [here](https://github.com/caphg/shop-lab-frontend)

## Stack
========

* Ruby on Rails 4.2.3

- API implemented with `rails-api` gem and `active_model_serializers`
- Token authentication via `devise_token_auth`

* PostgresDB
